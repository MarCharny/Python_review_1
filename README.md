# Python_review_1
## Основная информация
В проекте реализована игра Тамагочи, где пользователю предлагается выбрать питомца и ухаживать за ним.

## Особенности запуска
В реализации игры используется модуль **pygame**

Инструкцию по установке можно увидеть на [официальном сайте](https://www.pygame.org/wiki/GettingStarted)

Так же в проекте присутствует множетсво графических файлов. Для корректного запуска все они должны лежать в одной папке с файлом .py
